Smart Rain Alert System

This is an IoT project that creates a rain alert system using ESP8266, Arduino UNO, and various sensors. ESP8266 sends data to the Blynk application via the Internet, while Arduino UNO collects data from sensors (Water Level Sensor and DHT11 Sensor) and sends it to ESP8266, which then uploads the data to Blynk for rain alert notifications.

Traffic Light

This digital project simulates a road crossing system with safety barriers that work in conjunction with automatic traffic lights. The traffic light circuit is designed using various ICs connected to an Arduino UNO, which controls the barrier based on traffic light signals. When a switch is pressed, the traffic light changes from green to yellow and then to red, and if the light is red, the Arduino UNO commands the barrier to block the road.

Aqua Vision

This project involves measuring and controlling water levels using image processing technology. It uses a Raspberry Pi Zero W for processing and analyzes video data from a camera to monitor water levels. If the water level exceeds a set threshold, the system commands a pump to operate automatically, with the Arduino UNO working in tandem with the Raspberry Pi Zero W.
